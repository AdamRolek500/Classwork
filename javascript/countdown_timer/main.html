The countdown timer will default to the following new year from the current date. I wanted to add
more functionality as a challenge, so I have added the ability to control the target date. You
must choose a date and specify a time. After clicking submit the target date will change if the
resulting date object is valid. As a style choice I have included the current and future time in
the date strings. The current time and date will update every second as well
as the remaining time.
